# Home Decor Website

A simple static website showcasing a home decor design using HTML and CSS.

## Project structure

- `index.html` - main page markup
- `style.css` - page styling
- `home-images/` - image assets used in the site

## How to run locally in VS Code

1. Open the folder `e:\projects\Home-Decor-Website` in Visual Studio Code.
2. Open `index.html`.
3. Recommended: install the `Live Server` extension.
   - Right-click `index.html` and choose `Open with Live Server`.
   - The site will open in your browser and refresh automatically when you save.

### Alternative local server

If you prefer using the built-in terminal:

- Open the terminal in VS Code (`Ctrl+``)
- Run:
  - `python -m http.server 8000`
- Open `http://localhost:8000`

## Notes

- No build or install step is required.
- This is a static HTML/CSS project.
